# MLIT Photo Pro ユーザーガイド

本プロジェクトをご利用いただき、ありがとうございます。
フォルダ構成が整理され、より使いやすくなりました。

## フォルダの歩き方

- **`/app`**: **アプリ本体**です。
  - プログラム（ソースコード）や設定ファイルが含まれています。
  - そのまま ZIP 圧縮して配布・ダウンロードすれば、他の環境でも開発・ビルドが可能です。
- **`/dev_tools`**: **開発者向けツール**です。
  - 最新の要領基準を公式から取得したり、各種データをパースするためのスクリプトが入っています。一般の利用者は通常触る必要はありません。
- **`/tests`**: **テスト用資材**です。
  - 動作確認に使用できるサンプルの写真データなどが含まれています。

## 使い始めるには

### 1. 準備
Node.js がインストールされていることを確認してください。

### 2. 起動（開発モード）
ルートディレクトリで以下のコマンドを実行します。
```bash
npm install
npm run dev
```
ブラウザが起動し、写真管理画面が表示されます。

### 3. ビルド（配布用ファイルの作成）
Webサーバー等に載せて公開したい場合は、以下のコマンドを実行します。
```bash
npm run build
```
`app/dist` フォルダに、公開用の HTML/JS/CSS ファイルが生成されます。

## 注意事項
- **ローカル完結**: 入力した写真や工事情報は、外部サーバーにアップロードされることはありません。すべてお使いのブラウザ内（ローカル）で処理されます。
- **保存**: 「XML生成・保存」ボタンを押すことで、指定したフォルダに国土交通省基準準拠の `PHOTO.XML` 等が書き出されます。
