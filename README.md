# MLIT PHOTO PRO (v0.2)

国土交通省「デジタル写真管理情報基準」に準拠した写真管理・PHOTO.XML生成ツールです。

## 特徴
- **完全ローカル動作**: ブラウザの File System Access API を活用し、データをすべてローカル内で処理するため、機密性の高い工事写真も安全に管理できます。
- **モダンアーキテクチャ**: Vite + React + TailwindCSS ベースにより、高速で洗練されたUXを提供します。

## プロジェクト構成

本リポジトリは、一般ユーザー向けの「アプリ本体」と、開発者向けの「保守ツール」に分離されています。

- **`app/`**: アプリケーション本体です。配布やダウンロードの対象となります。
- **`dev_tools/`**: 基準データの更新やパース用スクリプトなど、開発者向けのツール群です。
- **`tests/`**: 写真データなどのテスト用サンプル資材です。

### 開発とビルド

ルートディレクトリから以下のコマンドを実行できます。

```bash
# 開発サーバーの起動 (app/ 内で実行されます)
npm run dev

# アプリのビルド (app/dist/ に生成されます)
npm run build
```
- **あらゆる工事・業務基準に対応**:
  - デジタル写真管理情報基準（R05, R02, H28 等）
  - 土木工事・業務の電子納品要領（R06, R05, R04, R03 等）
  - 官庁営繕（Eizen）ガイドライン・要領
  - その他、港湾、電気通信、機械設備、地質調査、測量等の100件以上の基準に対応。

## 開発者向け (Maintenance)
- **基準データの同期・監査**: 開発者は `.agents/skills/sync-mlit-standards` を通じて、公式サイトから最新情報を取得し `standards.js` を更新できます。これは一般ユーザー向けの機能ではありません。

## 開発環境
- Node.js (v18以降推奨)
- `npm run dev`：開発サーバーの起動
- `npm run build`：本番ビルドの生成
